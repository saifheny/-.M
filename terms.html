1. مقدمة

مرحباً بك في منصة AI Agent Pro. باستخدامك لهذه المنصة، فإنك توافق على الالتزام بهذه الشروط والأحكام. يُرجى قراءتها بعناية قبل استخدام أي من خدماتنا.

منصة AI Agent Pro هي منصة ذكاء اصطناعي متكاملة توفر واجهة موحدة للتفاعل مع نماذج AI متعددة، ومحرر أكواد مدمج، وأدوات إنتاجية متقدمة.

2. حسابات المستخدمين

  • يجب أن تكون بالغاً (18 عاماً فأكثر) أو تحت إشراف ولي أمر لاستخدام المنصة
  • أنت مسؤول عن الحفاظ على سرية بيانات حسابك وكلمة المرور
  • يجب تقديم معلومات دقيقة وصحيحة عند التسجيل
  • يحق لنا تعليق أو إنهاء حسابك في حالة انتهاك هذه الشروط
  • الحسابات الزائرة (Guest) مؤقتة وقد يتم حذف بياناتها

3. الاستخدام المقبول

  • استخدام المنصة لأغراض مشروعة وقانونية فقط
  • عدم إنشاء محتوى ضار أو غير قانوني أو مسيء عبر نماذج الذكاء الاصطناعي
  • عدم محاولة اختراق أو إساءة استخدام واجهات API
  • عدم مشاركة مفاتيح API الخاصة بمقدمي الخدمات مع أطراف ثالثة
  • عدم استخدام المنصة لإرسال رسائل غير مرغوبة (Spam)
  • الالتزام بشروط استخدام مقدمي خدمات الذكاء الاصطناعي (Groq، OpenAI، Google، إلخ)

4. مفاتيح API والفوترة

  • المنصة لا تخزن مفاتيح API على خوادمها — جميع المفاتيح تُحفظ محلياً على جهازك
  • أنت المسؤول الوحيد عن أي تكاليف ناتجة عن استخدام مفاتيح API الخاصة بك
  • المنصة غير مسؤولة عن أي رسوم يفرضها مقدمو خدمات الذكاء الاصطناعي
  • يُنصح بمراقبة استهلاكك بانتظام من لوحات تحكم مقدمي الخدمات

5. إخلاء المسؤولية

  • المنصة تُقدم "كما هي" بدون أي ضمانات صريحة أو ضمنية
  • لا نضمن دقة أو صحة المحتوى المُولّد بواسطة الذكاء الاصطناعي
  • لا نتحمل المسؤولية عن أي أضرار ناتجة عن استخدام المنصة
  • الخدمة قد تتوقف مؤقتاً للصيانة أو لأسباب تقنية

6. الملكية الفكرية

  • جميع حقوق الملكية الفكرية للمنصة محفوظة لمؤسسة SA
  • المحتوى الذي تنشئه عبر المنصة يبقى ملكاً لك
  • لا يحق لك نسخ أو توزيع أو إعادة إنتاج كود المنصة بدون إذن
